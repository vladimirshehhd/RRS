# Этап 1. Описание проекта и границы

**Файл:** `docs/01_about.md`

## 1. Название проекта: 
**NewsRSS**

## 2. Что делает приложение
NewsRSS — это простой веб‑сайт с лентой новостей, который собирает публикации из одного или нескольких RSS‑источников и показывает их в удобном списке. Пользователь открывает сайт, видит свежие заголовки, может перейти к полной новости по ссылке и обновить ленту.

Основная цель — показать работу веб‑приложения с внешним RSS‑источником, отображением новостей и базовой фильтрацией.

## 3. Пользователь
Любой человек, которому нужно быстро посмотреть свежие новости из RSS в одном месте, сайт нужен для того, чтобы без лишних действий открыть страницу и сразу увидеть последние публикации.

## 4. Функции MVP
- Загрузка новостей из RSS‑ленты по кнопке или при открытии сайта.
- Отображение списка новостей: заголовок, краткое описание, дата публикации, источник.
- Переход к полной новости по внешней ссылке.
- Фильтрация списка по ключевому слову.
- Переключение между несколькими заранее заданными RSS‑источниками.
- Защита от дублей новостей по ссылке/идентификатору.

## 5. Что не делаем
- Не делаем регистрацию и авторизацию пользователей.
- Не делаем собственную систему публикации новостей через админ‑панель.
- Не делаем комментарии, лайки и социальные функции.
- Не делаем сложные рекомендации, теги на базе ML и персонализацию.
- Не делаем мобильное приложение.

## 6. Платформа / стек (предварительно)
Платформа: Web
Репозиторий и совместная разработка: GitHub
Язык: Python 3.11+
Бэкенд-фреймворк: Flask
Шаблоны: Jinja2
Парсинг RSS: feedparser
Хранение данных: SQLite
Стили: простой CSS без тяжёлых UI-библиотек
